# Styled System

Responsive, theme-based style props for building design systems with React

[![gh][]][github] [![stars][]][github] [![Build Status][ci]][travis] [![Downloads][]][npm] [![Version][]][npm] ![size][] ![MIT License][license]

- Add style props that hook into your own theme
- Responsive prop values for quickly setting responsive font-size, margin, padding, width, and more
- Influenced by constraint-based design system principles
- Typographic scale
- Spacing scale for margin and padding
- Works with any color palette
- Works with most CSS-in-JS libraries, including styled-components and emotion
- Used in Rebass, Rebass Grid, and the Priceline Design System
