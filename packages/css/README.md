# @styled-system/css

Style props for _all_ CSS properties

```sh
npm i @styled-system/css
```

```js
import styled from 'styled-components'
import css from '@styled-system/css'

const Box = styled.div(css)
```

```jsx
<Box
  color="blue"
  fontSize="32px"
  fontWeight="bold"
  padding="16px"
  boxShadow="0 0 0 32px rgba(0, 0, 0, .125)"
/>
```
